### Real Time, Forecasted, Future, Marine and Historical Weather

Free Weather Forecast in JSON and XML for commercial and non-commercial use

##### Real-time Weather API

Real-time or current JSON weather and XML weather API provides access to near real-time weather information including pollen for millions of locations worldwide by global collaborations of weather stations and high resolution local weather models.

[Learn more →](https://www.weatherapi.com/api.aspx)

##### Historical and Future Weather

14 day daily, hourly and 15 min interval weather forecast, future weather (upto 365 days ahead), solar irradiance, pollen, marine weather and hourly historical weather (from 2010 onwards) for millions of cities and towns around the world in JSON and XML format.

[Learn more →](https://www.weatherapi.com/api.aspx)

##### Geolocation API

Air Quality Data, Pollen, Time zone, Astronomy, IP lookup, Sports API, Currency API, Map API and Geo API.

[Learn more →](https://www.weatherapi.com/api.aspx)

Lightweight XML and JSON format with 200 ms average response time. Bulk request with complete weather API documentation.
